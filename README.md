A comprehensive **Personal Finance Management System** built with **Python** and **Streamlit** that helps individuals manage their income, expenses, loans (EMIs), savings, investments, financial reports, and AI-powered financial insights from a single dashboard.

---

# 📌 Project Overview

Managing personal finances effectively is essential for financial stability. This application enables users to record, analyze, and visualize their financial activities while receiving intelligent recommendations to improve spending habits, savings, and investments.

The system provides a centralized platform to monitor financial health through interactive dashboards, charts, analytics, and reports.

# 🖥 Installation

Clone the repository

```bash
git clone https://github.com/drknatarajan17/Smart-Personal-Finance-Manager.git
```

Move into the project

```bash
cd Smart-Personal-Finance-Manager
```

Install dependencies

```bash
pip install -r requirements.txt
```

Run the application

```bash
streamlit run app.py
```
